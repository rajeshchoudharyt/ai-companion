### AI Companion

A multi-agent system that understands and responds based on user memory and emotional patterns with a personality engine, memory and emotion extraction module autonomously.

**Features:**

1. Facts and preferences extraction from conversation.
2. User emotion extraction.
3. User emotional patterns analysis.
4. AI Personality engine with multiple personalities.
5. Automatic switch between different personalities mid-conversation based on uee emotional pattern.
6. Interactive UI that displays user memory, emotional pattern and selected personality.

***

### Steps to start

1. Install dependencies.

   ```pip install -r requirements.txt```

2. Run server

   ```cd app && fastapi run main.py```

***

Deployment url: https://ai-companion-frontend-seven.vercel.app/
